# Task Management (task-flow)

A simple task management web app (backend + frontend) built with Node.js, Express, MongoDB, and a Vite + React frontend.

## Repository layout

- `backend/` — Express server, controllers, models, routes
- `frontend/` — Vite + React single-page app

## Quick start

1. Backend: install and run

```bash
cd backend
npm install
npm run dev
```

2. Frontend: install and run

```bash
cd frontend
npm install
npm run dev
```

## Notes

- Ensure MongoDB is running and configured in `backend/config/db.js`.
- The repo was pushed to https://github.com/Modi6090/task-flow
